
# Quintanda Virtual

Aplicativo de loja virtual usando flutter e GetX


## Instalação

```bash
  $ git clone https://github.com/luizmineiro/quintanda_virtual.git

  $ cd quintanda_virtual

  $ flutter pub get

  $ flutter run
```
    
## Stack utilizada

**Front-end:** Flutter, GetX, Material Design

**Back-end:** Parse Server
